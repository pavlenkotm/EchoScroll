# ⚡ Zig Cryptography

[![Language](https://img.shields.io/badge/language-Zig-orange.svg)](https://ziglang.org/)

Modern, safe systems programming for Web3 cryptography.

## Features

- ✅ SHA-256 and Keccak-256 hashing
- ✅ Wallet generation
- ✅ Zero allocations where possible
- ✅ Compile-time safety

## Build & Run

```bash
zig build run
```
